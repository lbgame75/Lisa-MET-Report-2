## STRUCTURE BY INSTRUMENT

### CONNERS-3 OR CONNERS-4

**Opening paragraph**

Name the instrument and its purpose. State who completed each form and when (or the approximate timeframe). If data from a rater is missing or incomplete at time of report, state that plainly. If the student has a relevant medical diagnosis, frame the Conners section as examining the educational implications of that condition — do not state the diagnosis as new information here.

Example opener when medical context applies:
> [Student] has a medical diagnosis of ADHD. The educational implications of his medical condition were examined to identify how the ADHD condition may be limiting his access to or adversely impacting his ability to progress in the general education curriculum.

Example opener when no medical context:
> [Student]'s parent and [number] of his teachers were asked to complete the Conners Rating Scales. Results are reported below.

**Score table**

Present the scale scores exactly as provided in a clean table. Format:

| Scale | Parent | Teacher (Subject) | Teacher (Subject) |
|---|---|---|---|
| Inattention | | | |
| Hyperactivity/Impulsivity | | | |
| Learning Problems/Executive Functioning | | | |
| Defiance/Aggression | | | |
| Peer Relations | | | |

Include the T-score classification key immediately below or above the table:

| T-Score Range | Classification |
|---|---|
| 70 and above | Clinically Significant |
| 60–69 | At-Risk |
| 41–59 | Average |
| 31–40 | Low |
| 30 and below | Very Low |

**Overall impression paragraph**

State which scales were clinically significant and from which rater(s). Do not build to this — lead with it. If one rater's results are substantially different from others, state that directly (e.g., "Results suggest [student]'s ADHD-related symptoms are more evident at home than in the school setting.").

**Item-level frequency check — do this before writing any scale paragraph**

If the computer-generated report includes an item analysis or item response table, scan it before writing. Identify every item rated at the highest frequency level (typically "Almost Always," "Very Often," or equivalent — the top rating on the scale). For each of those items:

1. Identify which content scale that item loads on.
2. Check whether that scale is Clinically Significant or At-Risk.
3. Use that to determine how to handle the example in the write-up:
   - **Clinically Significant scale:** Include the specific behavior as a concrete example in the paragraph for that scale. State it in plain language — describe the behavior, not just the category. These are the most important data points in the report.
   - **At-Risk scale:** Note the specific behavior but frame it as an area warranting monitoring rather than a confirmed clinical concern.
   - **Average scale:** Do not include the item example in the report unless it provides meaningful context for understanding a discrepancy across raters. Chart by exception.

If the computer report does not include item-level frequency data, skip this step and note in the section that item-level data was not available in the report at the time of writing.

**Scale-by-scale interpretation — elevated scales only**

Write one paragraph per elevated area. For each:
- State the scale name and classification level
- Describe the behavioral pattern in plain language (what this score range means behaviorally)
- Pull specific highest-frequency item examples (see item-level check above) and state them concretely — what the rater actually endorsed, in the rater's own words where possible
- If raters agree on an area, say so and integrate both; if they diverge, explain the difference

Do NOT write a paragraph for a scale in the average range unless it provides meaningful context for understanding a divergent pattern.

**Strengths noted by raters**

If any rater noted strengths, include them. Use direct quotes. Do not smooth them into paraphrase — the rater's exact words carry more weight than a summary.

Example:
> [Student]'s history teacher reported he is "very personable." His math teacher identified his strengths as "smart and focuses when by self." His mother reported he is "a good athlete, hard worker when given a job to do, shows empathy, has a good sense of humor."

**DSM symptom scales (Long Form only)**

If the Long Form was used, include the DSM Symptom Scales results. State which classifications were clinically significant and for which rater. If the ADHD Index was computed, report it and note the similarity score (e.g., "Similar scores were obtained by individuals with ADHD 82% of the time").

**Frequency question (if reported)**

If the computer report includes the rater's response to the global frequency question about how often the student's problems seriously affect functioning in academic and social settings, include it. State the rater's response directly.

Example:
> Both the parent and teacher reported "Very Frequently" when asked if [student]'s problems seriously affect his functioning in the academic setting, and "Occasionally" in the social setting.

### ASRS (AUTISM SPECTRUM RATING SCALES)

**Opening paragraph**

Name the instrument, its age range, and its purpose. State who completed the forms and their relationship/context to the student. Note if a rater has limited observation time (e.g., "the teacher who has been working with him for two weeks") — this is clinically relevant and must not be omitted.

Example:
> Based on ratings from [student]'s mother and the teacher that has been working with him for two weeks, the Autism Spectrum Rating Scale Total Score was in the Very Elevated range...

**Score table**

Present all scores in a clean two-column table (Parent T-Scores / Teacher T-Scores):

| Scale | Parent T-Score | Teacher T-Score |
|---|---|---|
| TOTAL SCORE | | |
| **ASRS SCALES** | | |
| Social/Communication | | |
| Unusual Behaviors | | |
| Self Regulation | | |
| **DSM-V SCALE** | | |
| DSM-V Scale | | |
| **TREATMENT SCALES** | | |
| Peer Socialization | | |
| Adult Socialization | | |
| Social Emotional Reciprocity | | |
| Atypical Language | | |
| Stereotypy | | |
| Behavioral Rigidity | | |
| Sensory Sensitivity | | |
| Attention | | |

Include the T-score classification key:

| T-Score | Classification |
|---|---|
| 70 and above | Very Elevated (Many more concerns than typically reported) |
| 65–69 | Elevated (More concerns than typically reported) |
| 60–64 | Slightly Elevated (Somewhat more concerns than typically reported) |
| 40–59 | Average (Typical levels of concern) |
| Less than 40 | Low (Fewer concerns than typically reported) |

**Total Score and DSM-V Scale paragraph**

State the Total Score classification for each rater. State the DSM-V Scale classification for each rater. State the conclusion plainly: whether these scores are consistent with ASD symptom presentation.

**ASRS Scales paragraph**

Cover Social/Communication, Unusual Behaviors, and Self-Regulation in a single paragraph. For each:
- Provide a brief plain-language descriptor of what the scale measures (one phrase or clause, written for a parent reader)
- State the classification range
- Integrate both raters naturally

Example descriptor format (from Lisa's sample):
> Social/Communication — the extent to which [student] uses verbal and nonverbal communication appropriately to initiate, engage in, and maintain social contact

Write this as running prose, not a bullet list.

**Treatment scales paragraph**

Identify the areas where both raters reached clinically significant agreement. Name each area. Describe the behavioral meaning in plain, school-based language — what the student actually does or does not do. Do not stop at category labels.

Example:
> Ratings suggest that [student] exhibits significant limitations in his willingness and capacity to successfully engage in activities that develop and maintain relationships with other youth and adults, has limited ability to provide an appropriate emotional response to another person in a social situation, and has limited ability to utilize spoken communication in a structured and conventional way.

**Suggested treatment targets**

List suggested treatment targets as bullet points, organized by domain. Pull these directly from the computer-generated report's elevated item analysis. Before listing targets, check whether the computer report includes item-level frequency data. If it does, identify items rated at the highest frequency level (Almost Always or equivalent) and confirm which treatment scale they load on. Include only targets that correspond to a Clinically Significant or Elevated treatment scale score. If the corresponding treatment scale is Average, do not include those targets. This is the one section in Lisa's MET reports where bullets are appropriate.

Example format:
> Suggested treatment targets for [student] based on elevated item ratings from both parent and teacher are listed below:
>
> **Peer Socialization**
> - Increase the ability to seek out other children for socialization
> - Increase the ability to greet/communicate with other children
>
> **Social Emotional Reciprocity**
> - Improve the ability to understand the feelings of others

## SCORE CLASSIFICATION GUIDE

**Conners-3/4:**
- T-score 70+: Clinically Significant
- T-score 60–69: At-Risk
- T-score 41–59: Average
- T-score 31–40: Low
- T-score 30 and below: Very Low

**ASRS:**
- T-score 70+: Very Elevated
- T-score 65–69: Elevated
- T-score 60–64: Slightly Elevated
- T-score 40–59: Average
- T-score below 40: Low

## VOICE RULES FOR THIS SECTION

**Do this:**
- Lead with the finding, not with the instrument description
- Name specific behaviors — what the rater actually checked or endorsed, not just the scale name
- Use direct quotes from raters when available; do not smooth them
- State rater agreement plainly and name which areas converge
- State divergence between home and school directly if it exists
- Keep plain-language descriptors of what each ASRS scale measures — one phrase, written for a parent
- For the ASRS treatment targets, use bullets — this is appropriate here

**Do not do this:**
- Do not write a paragraph for an average-range scale unless it explains a meaningful pattern
- Do not hedge clinically significant findings
- Do not use "taken together" or "overall" as a wrap-up
- Do not use em dashes
- No X/Y contrast constructions
- Do not inflate Low Average scores into concern language, and do not soften Clinically Significant scores into mild language
- Do not invent behavioral examples — only use what the computer report or rater provided
- Do not close with a generic summary paragraph — the section ends when the data ends
