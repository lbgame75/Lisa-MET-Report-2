Organize and Synthesize Educational Record into Narrative
WHEN TO USE THIS SKILL
You've reviewed the student's educational record and need to synthesize this into a coherent Educational History narrative.
WHAT THE USER WILL PROVIDE

Student name and grade/school
Schools attended and grades/years
Grade history
Attendance data
Academic performance trends
Teacher comments or concerns
Standardized test data
Discipline history
Prior special education services
Any other relevant information

YOUR JOB
Create a coherent, chronological narrative that:

Establishes the timeline
Describes academic performance trends
Integrates teacher voice and concerns
Notes patterns
Includes relevant data
States concerns plainly
Connects factors together

STRUCTURE
Opening paragraph: Establish the timeline

"This review spans [student]'s educational history beginning in [grade/year] and continuing through [current grade/year]."

Academic performance paragraphs: Grade trends and subject-level performance

"In [grade], [student] earned [grades/performance level] across core subjects."
Note patterns: improving, declining, variable, stable
Include teacher comments when they illuminate the picture

Attendance paragraph: If attendance is below 90% in ANY year

State percentages for each year
Define chronic absenteeism if applicable
Connect attendance to academic/behavioral impact

Standardized testing and benchmark data paragraph: If available

Overall pattern
M-STEP results
PSAT/SAT results
NWEA trends

Discipline history paragraph: ONLY if there is a significant behavioral history

State incidents plainly: what happened, consequences
Use a table if there are multiple incidents
Do NOT soften or minimize
Connect to academic/school functioning


HANDLING DISCIPLINE HISTORY
If significant behavioral history:

State what actually happened (not just category labels)
Include dates, descriptions, consequences
Do NOT soften language
SEVERITY RULE APPLIES: State behavior plainly and directly
Connect behavioral incidents to classroom/school functioning

HANDLING ATTENDANCE DATA
If attendance is 90% or above each year: Briefly note it
If attendance drops below 90%:

State specific percentages
Define chronic absenteeism: "Because attendance below 90% meets criteria for chronic absenteeism, [student] has experienced chronic absenteeism during [years], limiting [his/her/their] access to instruction."
Connect to impact

LISA'S VOICE RULES
DO THIS

Move forward in time naturally (chronological)
Integrate teacher voice without editorializing
State concerns plainly when data support them
Name specific schools, grades, years
Use active voice
Connect factors together

DO NOT DO THIS

No em dashes
No hedging: "appears to struggle"
No generic statements: "has had school challenges"
No softening of serious behavioral concerns
Do not treat Low Average grades the same as failing grades
Do not repeat information across paragraphs
