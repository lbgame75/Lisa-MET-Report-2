LISA'S VOICE RULES
DO THIS

Start with observable facts before moving to pattern interpretation
Use parallel structure for strengths and weaknesses
Include specific error examples
Name specific subtests and tasks
State patterns plainly: "This pattern suggests that..."
Connect each academic area to classroom functioning
State deficits plainly when the data support them

DO NOT DO THIS

No em dashes
No negative parallelisms: "it's not decoding, it's comprehension"
No short punchy pairs: "He can decode. Comprehension is the problem."
No performative setups: "What's worth examining is..."
No generic wrap-up sentences
No overpathologizing Low Average scores
No abstract language

SCORE CLASSIFICATION GUIDE

25th percentile and above: Average range
11th-24th percentile: Low Average
10th percentile and below: Below Average
6th percentile and below: Well Below Average

Do not overpathologize Low Average. State it plainly.
