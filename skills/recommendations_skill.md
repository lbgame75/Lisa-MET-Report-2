### RECOMMENDATION FORMAT

Each recommendation should:
1. **Start with a deficit-specific need**
2. **State the support concretely** (what to do, how to do it)
3. **Explain the "why"**

**Structure:** "[Student] [specific difficulty]. [Specific strategy/support]. This [reduces load / increases clarity / provides structure] and allows him to [positive outcome]."

**Good examples:**
- [Student]'s working memory limitations affect his ability to follow multi-step directions. [Student] learns best when directions are short, clear, and given one step at a time. This reduces the cognitive load.
- When reading grade-level material, [Student] would benefit from having the text read aloud or using text-to-speech, which allows him to focus on understanding instead of struggling to decode.
- In math, [Student] needs strong visual support and guided practice. Using models, manipulatives, and calculators for computation is recommended so that basic calculation does not prevent him from solving problems.

**NOT good:**
- [Student] would benefit from reading support. (too generic)
- [Student] needs Orton-Gillingham instruction. (too prescriptive)
- Increase time on task and build confidence. (not connected to profile)

**By Category:**

- Behavioral / Executive Function Support
- Sensory / Environmental Support
- Transition / Post-Secondary Planning (if applicable for 11th-12th grade or self-contained students)

**By Subject:**
- Reading
- Mathematics
- Written Expression
- Classroom Behavior / Executive Function

Choose whatever organization makes sense for THIS student.

### RECOMMENDATION CONTENT SCOPE

**What to include:**
- Classroom accommodations and modifications
- Instructional strategies
- Environmental supports
- Progress monitoring approach
- Communication with family
- Service intensity
- Teacher check-in / support for implementation

**What NOT to include:**
- Specific curriculum names (unless there's a specific reason)
- Medical interventions or medications
- Behavioral punishment or consequence systems
- Vague encouragement ("build confidence")
- Overly technical jargon

### INTENSITY AND REALISTIC IMPLEMENTATION

- A general education classroom teacher cannot implement 15 separate interventions
- Prioritize the 6-8 most important recommendations
- Group related recommendations together
- For a student with severe needs, recommendations might be heavier on resource/pullout instruction
- For a student with mild needs, recommendations might focus on classroom accommodations

### LISA'S VOICE RULES

#### DO THIS
- Ground every recommendation in the student's specific profile
- Use concrete, actionable language
- State the "why"
- Be specific about HOW to implement
- Prioritize
- Write for the teacher audience

#### DO NOT DO THIS

- No recommendations that don't connect to the evaluation data
- No contradictions
- No overly technical jargon
